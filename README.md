# E-commerce-platform

## Project Description

The E-commerce-platform project is an  e-commerce platform developed using C# .NET for the backend and React for the frontend.

## Architecture

As this project focuses on distributed systems (in order to complete the course), each microservice is stored in a separate repository. This architecture allows for easier management and development of individual components of the system.
